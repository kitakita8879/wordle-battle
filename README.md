# WORDLE BATTLE 雙人對戰

## 遊戲操作：
任意輸入由5個字母組成的單字。
輸入單字後，畫面會出現三種不同的顏色方塊。
  1.灰色代表答案內沒有該字母。
  2.黃色代表答案內有該字母，但位置不正確。
  3.綠色代表字母和位置皆正確。
雙人對戰時，最快猜中答案者勝利。
